## Git 提交建议

建议分为以下几个提交:

### Commit 1: 启用 BPF/BTF 内核支持
git add patches/main/linux-kernel-defconfig.patch
git add .tmp/bpf-btf-implementation-summary.md
git commit -m "feat: enable BPF/BTF kernel support

- Add 56 BPF/BTF configuration options
- Enable eBPF JIT compiler
- Enable XDP support
- Enable BPF LSM
- Enable BTF debug info generation

Required for Landscape eBPF router integration."

### Commit 2: 添加 Landscape 软件包
git add patches/vyos-build/0016-add-landscape-package.patch
git add scripts/build-landscape-package.sh
git commit -m "feat: add Landscape router package

- Create landscape-router deb package
- Download v0.18.3 binaries and static files
- Add systemd service configuration
- Add postinst/prerm scripts"

### Commit 3: 集成到构建流程
git add scripts/build-all.sh
git add scripts/patch-and-build-vyos-image.sh
git commit -m "feat: integrate Landscape into build process

- Add Landscape package build step
- Include landscape-router in custom packages
- Update build-all.sh workflow"

### Commit 4: VyOS 配置集成
git add patches/vyos-1x/vyos-1x-007-landscape-integration.patch
git commit -m "feat: add VyOS configuration for Landscape

- Add 'service landscape' configuration node
- Support enable/disable, port, and config-dir options
- Add Python configuration script
- Add XML interface definition"

### Commit 5: 文档
git add README.md
git add LANDSCAPE-INTEGRATION.md
git add LANDSCAPE-QUICKSTART.md
git add .tmp/landscape-integration-*.md
